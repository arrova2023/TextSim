Generación de Precisión y Cobertura

La API REST se utiliza para generar la precisión y la cobertura con un valor de corte que varía de 0 a 1. En este ejemplo la zona que mejor rendimiento genera corresponde al valor de corte n° 65 pero cambiando el contenido del Corpus el rendimiento va a variar.
